操作符

译者: 曾涛(未完成)

JavaScript有以下基本操作符

算术操作符

基本算术操作符包括：+, -, , /, % *[原著缺少乘号] 你可以将它们用在变量赋值当中就像下面的例子

var i = 2;
i = (2 + i) * 3 / i;
alert(i)

只是要注意, 整数相除会返回浮点数 这里用Java做对比 比如7/2 = 3(整数相除), 而JavaScript将返回给你一个正常的3.5 取模操作符 % 返回一个余数

i = 5 % 2; // 1
i = 8 % 3; // 2
i = 6 % 3; // 0

字符串连接

操作符 + 会连接字符串

var a = "my" + "string" // mystring
